Start with a Repo

I know, you're going to do this, but let's be sensible…

Create a new repo (with Github) https://github.com/new

  • Make sure your naming is Pascal Case - dotnet new uses the folder name for namespace
  • Tick Initialize this repository with a README - you'll want it sooner or later
  • Leave Add .gitignore as None
  • Add a licence - I like MIT
  • Press Create repository
  • git clone https://github.com/[your-name]/ReactCore.git

Getting Started

First steps first, open VS Code and navigate to your repo. Terminal -> New Terminal Ctrl+Shift+'

  • dotnet new react
  • npm install
  • npm audit fix
